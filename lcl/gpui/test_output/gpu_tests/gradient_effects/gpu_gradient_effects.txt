GPU Test: Gradient Effects
==========================

渲染内容:
1. 水平渐变: (100,100) -> (500,100), 红→蓝
2. 垂直渐变: (100,250) -> (100,350), 绿→黄
3. 对角渐变: (100,400) -> (500,500), 紫→橙
4. 圆角渐变: (600,100) -> (1000,100), radius=20
5. 彩虹渐变: (600,250) -> (1000,250), 红→蓝
6. 透明度渐变: (600,400) -> (1000,400), 透明→不透明
7. 径向渐变模拟: 中心(1200,300), 10层同心圆

验证标准:
- 渐变过渡平滑无色带
- 方向正确
- 圆角处无断裂
- 透明度渐变正确
