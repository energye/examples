GPU Test: Dynamic Animation
===========================

渲染内容:
1. 旋转正方形: 中心(200,200), 大小80, 旋转速度2rad/s
2. 脉冲圆: 中心(500,200), 半径30-50, 脉冲速度3Hz
3. 移动的点: 椭圆轨迹(700,200), 半径100x50
4. 波浪线: (100,400) -> (500,400), 振幅30, 频率0.05
5. 旋转星形: 中心(1000,200), 半径60, 旋转速度1rad/s
6. 渐变色动画: (1200,150) 200x100, 颜色循环

验证标准:
- 所有动画流畅无卡顿
- 边缘平滑无锯齿
- 颜色过渡自然
- 运动轨迹正确
