以下是基于 Ant Design 5.x 控件库整理的全量组件产品需求清单，按**通用、布局、导航、数据录入、数据展示、反馈、其他**分类。每个控件从**显示效果、互动效果、功能**三个维度描述，并附带关键状态与变体，用于指导基于 Go 语言 UI 渲染库实现一套类 Ant Design 的组件体系。

---

## 1. 通用

### Button 按钮
- **显示效果**  
  提供 `primary`、`default`、`dashed`、`text`、`link` 等类型；可通过 `danger` 切换危险样式；支持 `ghost` 幽灵按钮（深色背景上使用）。形状支持 `round`（圆角）与 `circle`（圆形）。尺寸分 `large`、`middle`、`small`。可内嵌图标，图标可居左或居右。`block` 属性使按钮撑满父容器宽度。禁用态整体降低透明度并显示禁止光标。
- **互动效果**  
  悬停时背景色与边框色按主题变量变化；点击时出现向内阴影或缩放动效（可含涟漪扩散）。加载中状态显示 loading 图标，按钮文字可保留或隐藏，同时禁用点击。获得焦点时展示轮廓（outline）。
- **功能**  
  点击触发回调，支持配置防重复点击（可由 `loading` 或异步控制自动防抖）；支持键盘 `Enter`/`Space` 触发；可充当表单提交按钮，触发表单 `submit` 行为。

### FloatButton 悬浮按钮
- **显示效果**  
  固定在屏幕边缘（右下角等）的圆形或方形按钮。可包含图标或文字，支持 `shape`、`type` 等样式变量。可附着 Badge 显示未读数字。可展开为悬浮按钮组（多个子按钮垂直/水平排列）。
- **互动效果**  
  悬停、点击动画与 Button 类似；展开/收起时子按钮有位移与透明度过渡。支持拖拽调整位置（可选）。
- **功能**  
  常用于返回顶部、全局操作入口等。支持点击事件，可配置触发行为。可在移动端与 PC 端使用不同的默认位置。

### Icon 图标
- **显示效果**  
  渲染矢量图标（SVG 或字体图标），支持通过 `size`、`color`、`rotate`、`spin` 控制外观。
- **互动效果**  
  若设置 `spin` 则持续旋转。可配合按钮等获取点击态颜色继承。
- **功能**  
  提供预设的图标库；允许自定义图标组件。支持两色图标（非必须）。

### Typography 排版
- **显示效果**  
  提供 `Title`、`Text`、`Paragraph` 等组件，控制字体大小、行高、字重、颜色、下划线、删除线、代码样式、标记高亮等。支持省略号截断（单行/多行）。
- **互动效果**  
  可复制：点击复制按钮或选中文本后出现复制成功提示；可编辑：点击文本切换为输入框编辑，失焦或回车保存。
- **功能**  
  文本复制、编辑、省略溢出处理。支持链接样式，点击跳转。

## 2. 布局

### Divider 分割线
- **显示效果**  
  水平或垂直分割线，可实线/虚线。支持嵌入文字（文字居中、左对齐、右对齐），可设置文字方向（`orientation`）。
- **互动效果**  
  无特殊交互。
- **功能**  
  内容分隔，纯展示组件。

### Flex 弹性布局
- **显示效果**  
  对子元素进行 flex 排版，支持 `justify`、`align`、`gap`、`wrap` 等属性可视化设置。
- **互动效果**  
  无。
- **功能**  
  替代手写 flex 样式，通过属性快速搭建弹性布局。支持响应式断点调整方向与间距。

### Grid 栅格
- **显示效果**  
  24 列栅格系统。`Row` 与 `Col` 组合，`Col` 支持 `span`、`offset`、`pull`、`push` 控制列宽与偏移。可设置 `gutter` 间距，显示为列间均匀空隙。
- **互动效果**  
  无。
- **功能**  
  响应式断点（`xs` `sm` `md` `lg` `xl` `xxl`），不同宽度可设不同列占比，实现弹性网格布局。

### Layout 布局
- **显示效果**  
  包含 `Header`、`Sider`、`Content`、`Footer` 区域。`Sider` 可折叠至窄条，宽度可调；可固定在视口一侧。内置触发按钮样式。
- **互动效果**  
  `Sider` 折叠/展开时伴随宽度过渡动画；触发器箭头旋转。可在移动端点击遮罩层关闭侧栏。
- **功能**  
  快速搭建带侧栏的页面框架。支持触发折叠、受控/非受控模式。

### Space 间距
- **显示效果**  
  在子元素之间插入均匀间距，可设置水平/垂直方向及 `size`（小、中、大或数值），支持 `wrap` 换行，对齐方式。
- **互动效果**  
  无。
- **功能**  
  便捷控制组件间间距，避免手写 margin。

## 3. 导航

### Affix 固钉
- **显示效果**  
  将元素固定在屏幕可视区域的某个位置（如顶部），占位保留原高度，防止内容跳动。
- **互动效果**  
  滚动页面达到设定阈值时，元素由相对定位变为固定定位，可能带有淡入或阴影效果。
- **功能**  
  监听滚动容器，配置 `offsetTop`/`offsetBottom` 触发固定。提供 `onChange` 回调通知固定状态变化。

### Breadcrumb 面包屑
- **显示效果**  
  显示层级路径，项之间用分隔符（可自定义）连接。支持图标、下拉菜单（用于同级跳转）、链接样式。
- **互动效果**  
  鼠标悬浮项时高亮；点击非最后一项可导航；如果某项含下拉菜单，悬浮或点击展示菜单。
- **功能**  
  与路由集成，自动生成面包屑。支持受控展开菜单。

### Dropdown 下拉菜单
- **显示效果**  
  点击或悬浮触发器后，浮层展示菜单列表，支持分割线、分组标题、图标、禁用项、危险项样式。可以包含自定义内容。
- **互动效果**  
  触发器点击或悬浮（可配置）弹出菜单，动画为自上而下展开并淡入；点击菜单项或外部区域关闭，关闭时反向动画。支持键盘上下选择、回车执行、Esc 关闭。
- **功能**  
  菜单项可绑定点击事件，支持多级菜单嵌套（SubMenu）。可自定义触发器元素。

### Menu 导航菜单
- **显示效果**  
  提供垂直/水平/内嵌模式。支持图标、分组、选中项高亮（主题色背景或文字色）、禁用项灰色。内嵌模式可展开/收起子菜单，前面有箭头指示。
- **互动效果**  
  悬停菜单项背景变化；点击父级项展开/收起子菜单（动画高度过渡）；选中项具有活跃样式。水平菜单下拉子菜单有弹出动画。键盘导航：方向键移动焦点。
- **功能**  
  支持 `selectedKeys`、`openKeys` 受控，支持路由联动。可配置 `inlineCollapsed` 适配折叠侧栏。

### Pagination 分页
- **显示效果**  
  页码按钮、上一页/下一页、快速跳转输入框、总条数展示、每页条数选择器。支持小尺寸、简洁模式（仅显示页码和箭头）。
- **互动效果**  
  悬停页码高亮，点击页码切换当前页，伴随选中态变化；快速跳转输入框失焦或回车时跳转。禁用的箭头呈灰色不可点。
- **功能**  
  分页数据计算（总页数），支持受控 `current`、`pageSize`，事件回调。可配置页码显示数量、是否显示总条数等。

### Steps 步骤条
- **显示效果**  
  水平或垂直步骤指示器。每步有图标（序号或自定义图标）、标题、描述。状态分 `wait`、`process`、`finish`、`error`。完成步显示勾选图标，进行步高亮，错误步显示叉号且标题变红。支持点状步骤条。
- **互动效果**  
  若步骤可点击，悬停时出现可点样式；点击切换步骤（受控），过渡状态变化。
- **功能**  
  引导式流程展示，支持受控当前步。可集成表单验证状态。

## 4. 数据录入

### AutoComplete 自动完成
- **显示效果**  
  输入框加下拉面板，面板展示匹配的建议项列表，可高亮匹配字符。支持自定义选项渲染。
- **互动效果**  
  输入时下拉面板出现（过滤动画），键盘上下选择候选项，回车选中，Esc 关闭。悬停选项背景变化。
- **功能**  
  支持本地数组或远程搜索、防抖、自定义过滤规则。提供 `onSelect`、`onSearch` 回调。

### Cascader 级联选择
- **显示效果**  
  点击触发器弹出多层列表面板，每一列展示同级选项，选中某项后下一级选项在相邻列出现。选中项在触发器中显示路径文本（可自定义分隔符），支持多选（显示标签）。可搜索过滤选项。
- **互动效果**  
  触发器点击弹出面板（动画），悬停或点击父项展开子列；选中节点后自动收合（可配置）。多选时已选项在输入框中展示为 Tag，可单独删除。
- **功能**  
  支持受控值、动态加载子节点、变更回调、自定义字段别名。支持“可任意级选中”模式。

### Checkbox 多选框
- **显示效果**  
  勾选框加标签文本，可单独或成组。提供 `checked` 实心勾选、`indeterminate` 半选样式。支持卡片式按钮样式（Checkbox.Button）。禁用态灰显。
- **互动效果**  
  点击框或标签切换选中状态，伴随过渡勾选动画。组模式下全选/取消联动。悬停框边框高亮。
- **功能**  
  `Checkbox.Group` 管理多选值，支持全选。提供事件回调。

### ColorPicker 颜色选择器
- **显示效果**  
  触发器展示当前颜色块，弹出面板包含：预设色板、调色板（饱和度和明度选择）、色相滑块、透明度滑块、HEX/RGB/HSB 输入框。支持多种格式。
- **互动效果**  
  点击触发器弹出面板（带过渡动画）；拖拽调色板指示器实时预览颜色变化；滑块拖动更新值；输入框编辑；点击预设块选中。支持禁用态、触发器只读。
- **功能**  
  受控 `value`，颜色格式转换，推荐预设色，支持自定义面板触发方式。提供 `onChange` 完整颜色对象。

### DatePicker 日期选择器
- **显示效果**  
  输入框显示选中日期/时间格式文本，带有日历图标。弹出日历面板：月份切换按钮、日期网格，今天高亮，选中日期标记为蓝色圆背景。提供年/月/季度选择模式。范围选择器显示起止日期，中间日期区浅蓝色。
- **互动效果**  
  点击输入框或图标弹出日历动画；悬停日期格背景变化；点击日期选中并关闭面板。范围选择时，悬停预览范围高亮；支持键盘方向键切换日期，回车确认。时间选择器有滚动或输入时分秒。
- **功能**  
  日期时间值的解析和格式化；禁用日期（`disabledDate`）；快捷面板（今天、昨天、一周前）；范围限制。支持 `showTime` 集成时间选择。

### Form 表单
- **显示效果**  
  提供布局（水平、垂直、行内），`Form.Item` 包裹控件，展示标签、必填星号、校验错误信息（红色文本）、帮助文字。验证失败时输入框边框变红，并有错误提示动效（如抖动）。
- **互动效果**  
  校验失败字段自动获得焦点；提交时滚动到第一个错误项。输入时实时校验或失焦校验。
- **功能**  
  数据收集、校验规则定义、表单提交、重置。支持字段值监听、联动、动态增减表单项。提供 `Form.List` 数组字段管理。

### Input 输入框
- **显示效果**  
  带边框的文本输入域，可带前缀/后缀图标或文字，支持 `allowClear` 显示清除按钮（文本存在时），密码框显隐切换图标。`textarea` 模式显示多行文本区，可带字数统计。
- **互动效果**  
  聚焦时边框高亮为品牌色；悬停边框色变化；输入时清除图标出现/消失；密码图标点击切换密文/明文，图标变化。禁用态背景变灰。可设置最大长度并显示计数。
- **功能**  
  受控值，支持 `onPressEnter` 事件，输入法组合输入事件。提供 `addonBefore` / `addonAfter` 组合输入框。

### InputNumber 数字输入框
- **显示效果**  
  类似 Input，但仅限数字输入，有上/下步进按钮（可隐藏）。支持格式化为千分位、小数位数、百分比显示。
- **互动效果**  
  点击步进按钮增减数值，长按可连续增减；键盘上下键调整；聚焦时边框高亮；超出范围时边界限制。
- **功能**  
  范围约束（min/max）、步长、精度控制。支持 `formatter` 和 `parser` 自定义显示。

### Mentions 提及
- **显示效果**  
  文本输入框，输入 `@` 后弹出用户列表浮层，选项可显示头像和名称。选中后以特殊标签样式嵌入文本中。
- **互动效果**  
  键入 `@` 触发面板弹出，继续输入过滤选项；键盘上下选择，回车选中；选中的提及标记不可编辑删除，但可通过退格删除整个标签。
- **功能**  
  支持远程搜索，可自定义触发字符。受控 `value` 中提及为结构化数据。

### Radio 单选框
- **显示效果**  
  圆形选择框加标签，支持 `Radio.Button` 按钮样式（实心/描边风格），组内仅一个选中。禁用态灰显。
- **互动效果**  
  点击切换选中，动画过渡；按钮样式选中填充背景色。键盘方向键可在组内切换。
- **功能**  
  `Radio.Group` 管理值，支持 `optionType` 定义样式。事件回调。

### Rate 评分
- **显示效果**  
  星形图标序列，支持半星、自定义字符。选中星高亮（金色/主题色），未选中为灰色。可显示辅助文字。
- **互动效果**  
  鼠标悬停时预览评分（星临时点亮），点击确定评分；悬停移开后恢复选定值；可支持键盘方向键调整。允许清除（再次点击已选星）。
- **功能**  
  受控值，精度 0.5 或 1，只读模式。支持自定义图标。

### Select 选择器
- **显示效果**  
  触发输入框展示当前选中项（单选文本，多选显示Tag列表并折叠）。弹出下拉列表，选项可含图标、描述，分组标题，搜索输入框。下拉面板可虚拟滚动。
- **互动效果**  
  点击触发下拉动画；输入过滤选项；键盘导航选中项；多选时点击选项切换选中，选中Tag可删除。失焦关闭，可配置自动清除。
- **功能**  
  单选/多选，搜索过滤，远程加载数据，自定义渲染选项。支持 `optionLabelProp` 控制标签显示。

### Slider 滑动输入条
- **显示效果**  
  水平/垂直滑动轨道，已选轨迹着色，手柄（圆形或自定义）指示当前值。可带刻度标记、数字提示气泡。范围选择器显示两个手柄与选中区间高亮。
- **互动效果**  
  拖拽手柄滑动，显示Tooltip当前值；点击轨道跳转至该点。范围选择器手柄不可交叉。键盘左右键微调。
- **功能**  
  支持步长、范围限制、垂直模式、反向。`marks` 自定义标记点，受控值。

### Switch 开关
- **显示效果**  
  椭圆形容器，内含圆形滑块；开启状态背景为品牌色，滑块居右；关闭状态背景灰色，滑块居左。可带文字图标，小尺寸。
- **互动效果**  
  点击切换状态，滑块平移动画，背景色过渡。禁用时不可点击；加载状态可添加加载图标。
- **功能**  
  受控 `checked`，触发回调，支持 `loading`。

### TimePicker 时间选择器
- **显示效果**  
  输入框或触发器展示时间字符串，弹出面板含时、分、秒列（可滚动选择），或组合输入。支持 `showNow` 快捷选择当前时间。
- **互动效果**  
  点击弹出；滚动各列选择时间值，选中项高亮，点击确认关闭；键盘上下调整。与 DatePicker 结合使用时无缝切换。
- **功能**  
  步长定义、小时周期（12/24小时）、受控值。支持范围时间选择。

### Transfer 穿梭框
- **显示效果**  
  左右两列列表，左侧为数据源，右侧为已选。每列支持搜索框、全选复选框、分页，项显示标题和描述。移动按钮在中间。
- **互动效果**  
  选中项高亮，点击方向按钮将选中项移至另一侧并移除原列表，伴随过渡。支持拖拽排序已选列表（可选）。全选/取消。
- **功能**  
  受控目标键值、数据过滤、自定义渲染、懒加载。支持树形穿梭（Tree Transfer）。

### TreeSelect 树选择
- **显示效果**  
  触发器类似 Select，但下拉为树形结构，节点前有展开/收起箭头，可带复选框。选中项在触发器内显示完整路径或仅叶节点标签。
- **互动效果**  
  点击箭头展开/收起子节点；点击节点选中（单选或多选），若多选与父子联动；选中后下拉可保持打开或关闭。搜索时高亮匹配节点。
- **功能**  
  受控值、树数据异步加载、虚拟滚动、支持父子联动规则。

### Upload 上传
- **显示效果**  
  提供拖拽上传区域（虚线圈）、图片墙、卡片列表、按钮触发器等形式。文件列表显示进度条、文件名、大小、状态图标（上传中/成功/失败）以及删除和预览按钮。
- **互动效果**  
  点击触发文件选择，支持拖拽文件到热区（热区高亮），上传过程进度条实时更新；图片墙悬停出现遮罩和操作按钮；失败项可重新上传。
- **功能**  
  受控文件列表，前端校验类型、大小、数量；支持分片上传、上传前预处理、手动/自动上传；预览（图片、PDF等）。

## 5. 数据展示

### Avatar 头像
- **显示效果**  
  圆形或方形图片，若图片加载失败显示文字缩写；可按字符生成彩色背景头像。支持大小、形状、徽标。
- **互动效果**  
  若可点击，悬停有阴影或缩放；图片无特殊交互。
- **功能**  
  支持 `srcSet` 响应式图片、`onError` 处理；可作为组显示多个头像重叠。

### Badge 徽标数
- **显示效果**  
  在子元素右上角显示数字或状态圆点。数字可溢出显示 `99+`；状态点支持颜色；独立使用可直接显示文字。
- **互动效果**  
  无交互动画，但通常跟随子元素点击。
- **功能**  
  展示数量、状态；`count` 为 0 时可隐藏；支持 `overflowCount`。

### Calendar 日历
- **显示效果**  
  月视图网格，显示日期和内容。可高亮今天、选中日；头部可切换年月。可自定义日期单元格渲染，如添加日程标记。
- **互动效果**  
  点击切换月份箭头；点击日期触发选择；悬停日期背景变化。内置模式切换按钮（月/年）。
- **功能**  
  受控 `value`，`dateCellRender` 自定义内容。支持通知日程数据集成。

### Card 卡片
- **显示效果**  
  带边框或阴影的面板，含头部（标题、额外操作）、内容区、封面图、底部操作区。支持 `hoverable` 悬浮阴影加深；可设置 `loading` 时骨架屏。
- **互动效果**  
  悬停时阴影提升；头部操作按钮正常交互；网格型卡片有独立悬浮。可整卡点击。
- **功能**  
  内容容器，支持嵌套、列表布局 `Card.Grid`。提供选项卡模式（`Card.Meta`）。

### Carousel 走马灯
- **显示效果**  
  水平或垂直轮播容器，内容滑动切换，指示器显示当前位（圆点或数字）。渐变过渡或位移。可自动播放。
- **互动效果**  
  鼠标悬停暂停自动播放；左右箭头点击切换上一张/下一张（滑动动画）；指示器点击直达；触摸滑动支持。
- **功能**  
  自动播放间隔、动画效果、无限循环。受控索引。

### Collapse 折叠面板
- **显示效果**  
  多个面板项，每项有标题栏和展开/收起箭头，内容区可展开。可手风琴模式。标题可带额外元素。边框风格或无边框。
- **互动效果**  
  点击标题切换展开/收起，内容区高度动画展开；箭头旋转；禁用的面板不可点击。
- **功能**  
  受控 `activeKey`，手风琴模式一次只展开一项。自定义图标。

### Descriptions 描述列表
- **显示效果**  
  成组展示标签-值对，布局分水平（一行多列）和垂直。可设置 `bordered` 带边框表格样式，或普通分隔线。列宽按比例分配。
- **互动效果**  
  无。
- **功能**  
  响应式列数，支持嵌套复杂内容。可配合 `size` 控制紧凑度。

### Empty 空状态
- **显示效果**  
  居中展示大图标（自带或自定义）、标题文字、描述文字，以及底部操作按钮或链接。
- **互动效果**  
  无。
- **功能**  
  占位提示，可自定义图片和内容。

### Image 图片
- **显示效果**  
  显示图片，支持 `placeholder` 加载中占位和错误占位。可预览：点击后全屏遮罩展示大图，工具栏可缩放、旋转、切换上下张。
- **互动效果**  
  点击图片打开预览（渐入动画）；预览中缩放过渡、拖拽移动；键盘左右切换；点击遮罩关闭。
- **功能**  
  懒加载、`fallback` 错误兜底、多图预览组；支持自定义预览操作。

### List 列表
- **显示效果**  
  可带边框或分割线的纵向列表，每项可含头像、标题、描述、操作。支持虚拟滚动。`header` 和 `footer`。
- **互动效果**  
  悬停项背景变化；若项可点击，整个条目手型；分页加载滚动或点击“加载更多”。
- **功能**  
  数据驱动渲染，分页、加载状态、空状态。提供 `Item.Meta` 快速排版。

### Popover 气泡卡片
- **显示效果**  
  点击/悬浮触发器后弹出气泡浮层，可包含标题、内容和操作按钮。具有尖角箭头，支持12个方位。
- **互动效果**  
  触发时弹出动画（淡入和位移）；点击内部按钮或外部区域关闭；悬浮触发有移开延迟。焦点管理。
- **功能**  
  用于次要信息展示或简单操作（类似小对话框）。受控显隐。

### QRCode 二维码
- **显示效果**  
  渲染 Canvas/SVG 二维码图案，可带中间图标（如 Logo），支持状态：正常、过期、加载中（展示模糊加刷新）、扫描中（边框高亮动画）。
- **互动效果**  
  过期时可点击刷新按钮重新生成；无其他交互。
- **功能**  
  接收文本生成二维码，配置纠错等级、图标。支持 `status` 切换状态。

### Segmented 分段控制器
- **显示效果**  
  内联块级按钮组，选中项填充背景色滑动动画，项之间无间隙。支持块级、尺寸调整，可带图标和文字。
- **互动效果**  
  点击选项，选中态滑动切换（背景滑移，动画过渡）；禁用项不可点击。
- **功能**  
  受控值，类似 radio 按钮组，但视觉更紧凑。常用于视图切换。

### Statistic 统计数值
- **显示效果**  
  展示大号数值、标题，可带前缀后缀图标或单位，支持趋势箭头（上升/下降）和颜色。
- **互动效果**  
  数值变化时可添加数字跳动动画（`countUp` 效果）。
- **功能**  
  格式化数值，显示加载中骨架。倒计时模式（`Countdown`）。

### Table 表格
- **显示效果**  
  表格包含表头、数据行、边框、斑马纹、固定表头/列、可伸缩列宽。`loading` 时展示骨架或遮罩；空数据展示 `Empty`。支持行选择复选框、展开行、排序图标、筛选状态提示。树形数据缩进。
- **互动效果**  
  悬停行高亮；点击表头排序（箭头切换）；拖拽调整列宽；固定列阴影随横向滚动渐变；展开行动画；行选择联动。分页点击。
- **功能**  
  列定义、排序、筛选、分页、行选择、展开行、拖拽排序行、虚拟滚动、固定列、合并单元格、自定义渲染、行点击事件。

### Tabs 标签页
- **显示效果**  
  选项卡头部：`line`（下划线）、`card`（卡片）、`editable-card`（可关闭卡片）等。活动选项卡高亮，下划线滑动过渡。内容区切换。
- **互动效果**  
  点击选项卡滑动切换内容，`line` 下划线滑动动画；`card` 样式选项卡关闭按钮旋转；拖拽排序（可选）。超出时可滚动或折叠下拉菜单。
- **功能**  
  受控 `activeKey`，动态增删选项卡，路由联动。`tabBarExtraContent` 额外节点。

### Tag 标签
- **显示效果**  
  小圆角矩形，可带色彩、图标、关闭按钮。可勾选样式（`checkable`）。
- **互动效果**  
  可关闭标签点击关闭按钮消失（可选动画）；勾选标签点击切换选中填充；悬停背景变化。
- **功能**  
  视觉标记，颜色预设，受控显隐，事件回调。

### Timeline 时间轴
- **显示效果**  
  垂直时间线，左侧点（实心圆、空心圆或自定义），右侧内容，连线。可设置颜色区分状态（正常、成功、错误）。支持交替模式（左右内容）。
- **互动效果**  
  无特殊交互。
- **功能**  
  展示流程记录，支持自定义节点图标。可搭配 `pending` 节点展示加载。

### Tooltip 文字提示
- **显示效果**  
  悬浮时显示简单的文字气泡，背景黑色，文字白色，有三角箭头，可多方向。支持自定义颜色。
- **互动效果**  
  鼠标悬浮一定延迟后显示，移开或点击后消失（淡入/淡出动画）。焦点触发或点击触发可选。
- **功能**  
  简短的提示，支持受控显隐，`mouseEnterDelay` 等控制。

### Tour 漫游式引导
- **显示效果**  
  高亮目标元素，周围蒙层，气泡卡片指向目标，显示步骤标题、描述、操作按钮（下一步/上一步/完成），进度点指示器。
- **互动效果**  
  步骤切换时蒙层与气泡平滑位移至下一个目标；完成或关闭时蒙层淡出。点击蒙层可关闭（配置）。
- **功能**  
  引导流程，受控步骤，自定义高亮区域，滚动到目标。

## 6. 反馈

### Alert 警告提示
- **显示效果**  
  带背景色的横幅条，图标、信息、辅助文字，支持 `info`、`success`、`warning`、`error` 类型，可带有关闭按钮，可显示操作链接。顶部有边框强调色。
- **互动效果**  
  可关闭：点击关闭按钮，组件高度收缩消失动画。若包含操作按钮，正常交互。
- **功能**  
  全局或局部消息展示，支持 `closable`，提供关闭回调。可配 `banner` 模式。

### Drawer 抽屉
- **显示效果**  
  从屏幕边缘（左/右/上/下）滑出的浮层面板，背景有半透明遮罩。包含标题、关闭按钮、内容区和底部操作区。
- **互动效果**  
  点击触发遮罩或关闭按钮，面板滑出（带遮罩淡入）。打开时页面滚动锁住。关闭时面板滑回、遮罩淡出。`ESC` 关闭。
- **功能**  
  用于放置表单或详情，支持多种尺寸、自定义位置。受控显隐，`afterOpenChange`。

### Message 全局提示
- **显示效果**  
  页面顶部居中显示轻量信息条，自动消失。支持 info/success/error/warning/loading 图标样式，可带文字和关闭按钮。
- **互动效果**  
  出现时从顶部淡入并向下移动，持续数秒后淡出消失。多个提示依次堆叠。可手动点击关闭。
- **功能**  
  命令式调用（`message.success`），全局配置时长、最大显示数量。支持 `useMessage` hooks。

### Modal 对话框
- **显示效果**  
  居中或自定义位置的浮层弹窗，带有遮罩层。包含标题、内容、底部操作按钮（确定/取消）。可配置 `centered`。`confirm` 型有信息图标。
- **互动效果**  
  打开时弹窗从缩放淡入，遮罩淡入；关闭时相反。`ESC` 或点击遮罩可关闭（可禁止）；拖拽标题移动（可配）。确认加载时按钮显示 loading。
- **功能**  
  模态交互，支持异步关闭、footer 自定义、受控显隐、信息提示型快捷方法 (`Modal.info`)。支持多步骤、最大化。

### Notification 通知提醒框
- **显示效果**  
  屏幕右上角弹出通知卡片，含标题、内容、图标（类型色）、关闭按钮，可带操作。自动堆叠，支持展开详情。
- **互动效果**  
  从右侧滑入淡入，停留后或手动关闭时滑出淡出。可点击详情展开更多内容（高度过渡）。
- **功能**  
  非阻断式通知，命令式调用，配置持续时间、位置（四角）。支持 `useNotification`。

### Popconfirm 气泡确认框
- **显示效果**  
  类似 Popover 的气泡浮层，包含标题描述和“确定/取消”按钮，有三角箭头。
- **互动效果**  
  点击触发器弹出，点击确定或取消关闭；点击外部关闭。确定按钮可进入 loading 状态等待异步操作完成。
- **功能**  
  轻量确认操作，常用于删除等危险操作。受控显隐，`onConfirm` 回调。

### Progress 进度条
- **显示效果**  
  水平条状（填充色渐变）或环形/仪表盘，显示百分比数字或状态图标。成功/异常状态下颜色变化，异常可显示红色。`strokeColor` 可自定义渐变。
- **互动效果**  
  进度值变化时平移动画（宽度过渡）。环形进度条描边变化动画。动态格式支持自定义。
- **功能**  
  展示进度，受控 `percent`，步进、状态分段。支持小型进度条（内联）。

### Result 结果
- **显示效果**  
  居中显示状态图标（成功/失败/信息等预设），标题、副标题和操作按钮区。图标大尺寸，颜色主题。
- **互动效果**  
  无特殊交互。
- **功能**  
  提供标准结果反馈页，可自定义图标和文案。

### Skeleton 骨架屏
- **显示效果**  
  占位符块，模拟内容形状（矩形、圆形、段落），具有渐变闪烁动画。可组合形成复杂骨架。支持 `active` 动画。
- **互动效果**  
  仅展示，无操作。
- **功能**  
  加载占位，与真实组件尺寸对应，减少页面抖动。支持 `Skeleton.Input`、`Avatar`、`Button`、`Image`。

### Spin 加载中
- **显示效果**  
  旋转的加载图标（或自定义），可包裹内容使内容区半透明遮罩，居中显示加载符。内置不同大小。可带提示文字。
- **互动效果**  
  图标持续旋转；包裹模式内容不可交互（遮罩阻挡）。
- **功能**  
  全屏或局部加载状态指示，受控 `spinning`。支持 `delay` 延迟显示。

### Watermark 水印
- **显示效果**  
  在指定区域生成平铺的文字或图片水印，低透明度，斜向重复。覆盖于内容上方但不可交互（`pointer-events: none`）。
- **互动效果**  
  无。
- **功能**  
  页面防盗或版权信息，支持多行文字、自定义旋转角度、间距、字体。可动态更新内容。

## 7. 其他

### Anchor 锚点
- **显示效果**  
  固定在页面一侧的导航链接列表，表示页面滚动位置。当前激活项高亮，并有下划线或背景标记。支持多级嵌套。
- **互动效果**  
  点击链接平滑滚动到对应ID区域；页面滚动时自动更新激活项（通过监听滚动）。Tooltip 可显示标题。

### ConfigProvider 全局化配置
- **显示效果**  
  无具体UI，为后代组件提供统一的主题（颜色、字体、圆角等）、国际化文案、组件尺寸、`prefixCls` 等上下文。
- **互动效果**  
  无。
- **功能**  
  配置全局参数，动态切换主题（暗黑模式）、语言包。必须作为根容器包裹应用。支持 `theme` 令牌覆盖。

## 不限于以上控件能力的扩展需求（产品级）

1. **主题定制**：所有控件需支持通过主题令牌（token）修改主色、圆角、边框、阴影、字号、间距等，并支持暗黑模式一键切换。
2. **国际化**：日期、时间、分页、上传等文本需可通过全局配置切换语言。
3. **无障碍**：控件需支持 ARIA 属性，键盘操作（Tab、Enter、Esc、方向键），屏幕阅读器可识别状态变化。
4. **响应式与自适应**：栅格、表格、布局、弹窗等需考虑移动端触摸交互和视口适配，FloatButton、Drawer 等有移动特化行为。
5. **性能**：大数据量组件（Select、Table、Tree、Transfer）需要虚拟滚动/懒加载支持；动画使用 GPU 加速。
6. **状态管理**：所有录入控件必须支持受控和非受控两种使用模式，提供清晰的值变更语义。
7. **组合性**：布局类（Flex、Space、Grid）和排版类（Typography）应以声明式组合为第一原则，减少样式覆盖需求。
8. **Go 渲染库适配**：要求组件 API 采用结构化配置（如 struct 定义属性），事件回调注册，避免直接操作 DOM，同时提供声明式构建方法。渲染输出可根据目标（桌面原生、Web 等）适配相同交互逻辑。

这份清单覆盖了 Ant Design 5.x 的核心控件及其显示、互动与功能需求，可作为 Go 语言 UI 渲染库实现一套现代化企业级控件库的参考基线。
